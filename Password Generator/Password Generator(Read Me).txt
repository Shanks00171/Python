=======================Welcome to this password Generator=======================

This is a simple GUI Password Generator that generates random passwords with all the options that you select.

The user is asked to enter the length of the password they want, default value is 12.

The options include
	UpperCase - [A_Z]
	LowerCase - [a-z]
	Numbers - [0-9]
	Special Charecters - [!@#$%^&*()-=[];,./`_+:"<>?~]

The application uses random module of Python to select random chareters from these options and generates the password accordingly.

There is a Copy button which copies the generated password directly into your clipboard so that it can be easily pasted wherever needed.



[27/5/2021] - The Input Field stays at the center in both full screen and starting screen.
